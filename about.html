Welcome to Windy, a weather app developed as part of a group project at the University of Greater Manchester. This project leverages various modern technologies and practices to deliver a reliable and scalable weather forecasting platform.

Key Technologies and Services:

  • AWS (Amazon Web Services): Our platform is hosted on AWS using Amplify, Lambda for serverless computing, S3 for storage, Cognito for sign-ups & sign-ins, API Gateways for Connectivity, SNS for alerts, DynamoDB for Database.
  • CI/CD (Continuous Integration/Continuous Deployment): We use GitHub Actions & AWS Codepipeline to automate testing, building, and deployment, ensuring faster and more reliable updates.
  • Lambda API Integration: The app fetches real-time weather data via API calls to AWS Lambda, ensuring dynamic content delivery.
  • Responsive Design: The app is fully responsive and optimized for all screen sizes, providing an excellent user experience on both desktop and mobile.
  • User Authentication: We have integrated a simple authentication system using lambda functions for managing user's access.

Why This Project?

The goal of this project was to provide an intuitive, easy-to-use weather forecasting tool while incorporating modern technologies like serverless computing with AWS and automating our workflows using CI/CD practices.

We believe that Windy will not only help users plan their day better but also give us an opportunity to apply our learning in cloud computing, continuous integration, and modern web technologies.
